# nuxt-koa

> 基于nuxt-koa ssr服务端渲染

## Build Setup

``` bash
# install dependencies
$ npm run i  ||   npm run i xxx --save

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```
